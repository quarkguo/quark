Article REST API

Get a list of article basic information
http://localhost:8080/ccgrest/rest/article/basicInfo/listAll

Get article category by article ID
http://localhost:8080/ccgrest/rest/article/{articleId}/category

Get article content by articleID
http://localhost:8080/ccgrest/rest/article/{articleId}/content

Get category content by contentID
http://localhost:8080/ccgrest/rest/content/{contentId}

Get category content by categoryId
http://localhost:8080/ccgrest/rest/category/{categoryId}/content

Get article meta data
http://localhost:8080/ccgrest/rest/article/{articleId}/metadata

Create or Update article meta data, use POST please
http://localhost:8080/ccgrest/rest/article/metadata

Upload file

http://localhost:8080/ccgrest/upload_article.html

Indexing

Re-Indexing all articles
http://localhost:8080/ccgrest/rest/indexing/article/all

Index an article
http://localhost:8080/ccgrest/rest/indexing/article/{articleId}

Searching....

http://localhost:8080/ccgrest/rest/search?query=queryString&limit=10
