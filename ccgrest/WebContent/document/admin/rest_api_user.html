User REST API

Logout
http://localhost:8080/ccgrest/rest/user/logout

Get User Profile
http://localhost:8080/ccgrest/rest/user/profile

Get User List
http://localhost:8080/ccgrest/rest/admin/user/all

Get User by Id
http://localhost:8080/ccgrest/rest/admin/user/{id}

Get usergroup list
http://localhost:8080/ccgrest/rest/admin/usergroup/all

Get usergroup by id
http://localhost:8080/ccgrest/rest/admin/usergroup/{id}

Update Profile
http://localhost:8080/ccgrest/rest/user/updateProfile

Update Password
http://localhost:8080/ccgrest/rest/user/updatePassword
